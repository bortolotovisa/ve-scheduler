# VE Programming Scheduler

## Deploy to Railway (5 minutes)

### Step 1 — Create GitHub repo
1. Go to github.com → New repository → name it `ve-scheduler`
2. Upload all files from this folder (drag & drop in GitHub)
3. Click "Commit changes"

### Step 2 — Deploy on Railway
1. Go to railway.app → Sign up with your GitHub account
2. Click "New Project" → "Deploy from GitHub repo"
3. Select `ve-scheduler`
4. Railway detects Node.js automatically and deploys
5. Click "Generate Domain" → you get a link like `ve-scheduler-xxxx.up.railway.app`

### Step 3 — Share the link
Send the link to your team. That's it!

---

## Install as App (PWA)

**On Windows/Mac (Chrome or Edge):**
1. Open the link in Chrome or Edge
2. Click the install icon (⬇) in the address bar — OR click "Install App" button in the header
3. App appears in taskbar / Start Menu / Applications

**On iPhone:**
1. Open in Safari → Share button → "Add to Home Screen"

**On Android:**
1. Open in Chrome → three dots menu → "Add to Home Screen"
